# 3D 塔罗牌花活增强 Spec

## Why
塔罗牌项目已有不错的基础交互（手势抽牌、粒子系统、3D 卡牌），但缺少一些能让用户"哇"出声的惊喜感和深度玩法。需要增加几个独立的花活功能来提升整体体验的沉浸感和趣味性。

## What Changes
- **正逆位机制**：每张牌翻面时有 50% 概率为逆位（倒转 180°），逆位有独立的关键词解读，增加占卜的真实感
- **手绘魔法阵**：握拳状态下手掌在摄像头前画圈，3D 场景中同步生成旋转的魔法阵粒子特效（类似在空中画魔法阵）
- **三张牌阵模式**：抽完第一张后可选择继续抽"过去 / 现在 / 未来"三张牌，牌阵横向排列展示
- **Web Audio 音效**：加入简洁的音效反馈（握拳嗡鸣、翻牌音、牌堆脉冲、背景环境音），用 Web Audio API 合成，无需外部音频文件

## Impact
- Affected specs: 3D 手势塔罗牌交互
- Affected code: `projects/particles/index.html`

## ADDED Requirements

### Requirement: 正逆位机制
牌面翻转时随机判定正逆位，逆位卡牌视觉倒置并显示逆位关键词。

#### Scenario: 抽牌判定正逆位
- **WHEN** 用户完成抽牌手势，牌开始翻转动画
- **THEN** 牌有 50% 概率被判定为逆位，逆位时牌面最终旋转角度为 `Math.PI * 3`（即多翻半圈），牌面图像倒置

#### Scenario: 逆位显示
- **WHEN** 牌被判定为逆位
- **THEN** 结果面板标题下方显示"⬆ 正位"或"⬇ 逆位"标签，关键词和含义使用对应正/逆位的文案

### Requirement: 手绘魔法阵
握拳后手掌缓慢画圈，在 3D 场景中生成旋转的魔法阵粒子光环。

#### Scenario: 画圈检测
- **WHEN** 用户处于握拳状态且手掌做圆周运动（持续 > 1 秒）
- **THEN** 在 3D 场景中心生成旋转的魔法阵粒子环，跟随手掌画圈节奏旋转

#### Scenario: 画圈结束
- **WHEN** 用户停止画圈（松手或速度骤降）
- **THEN** 魔法阵粒子向四周散射消散，回归正常粒子状态

### Requirement: 三张牌阵模式
抽完第一张后提示用户可选择抽取第二、第三张牌，组成"过去 / 现在 / 未来"三张牌阵。

#### Scenario: 继续抽牌提示
- **WHEN** 第一张牌结果展示完成
- **THEN** 结果面板底部出现"再抽一张"按钮（最多可抽至 3 张）

#### Scenario: 三张牌展示
- **WHEN** 用户抽完 3 张牌
- **THEN** 结果面板改为三栏布局，分别标注"过去"、"现在"、"未来"，每栏显示对应牌的名称、正逆位和关键词

### Requirement: Web Audio 音效
使用 Web Audio API 合成简洁音效，增强操作反馈感。

#### Scenario: 握拳反馈
- **WHEN** 用户握拳成功
- **THEN** 播放低频嗡鸣音（~120Hz，0.3s 衰减）

#### Scenario: 翻牌反馈
- **WHEN** 牌面翻转动画开始
- **THEN** 播放上升音阶扫频音（200Hz→800Hz，0.5s）

#### Scenario: 背景环境音
- **WHEN** 进入抽牌阶段
- **THEN** 循环播放极低频环境嗡鸣（~55Hz，音量极低），增强沉浸感
