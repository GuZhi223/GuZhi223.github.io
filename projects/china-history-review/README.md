# 中国近代史答题式复习 PWA

## 已实现

- 74 道合并、去重、纠错后的复习题
- 按章节筛选、关键词搜索、随机抽题
- 自己输入答案后展开参考答案
- 掌握 / 模糊 / 错题 / 未学习四种状态
- 题目拆解、补漏提示和错题回流
- 可选 AI 补漏助手：仅在模糊题 / 错题后诊断缺漏，不替代自己作答
- 自动保存当前题目、作答内容和学习状态
- 导出与导入 JSON 进度备份
- PWA 安装到手机桌面
- Service Worker 离线缓存
- 响应式手机界面与深色模式

## 本地预览

PWA 的 Service Worker 不能直接在 `file://` 下运行。请在项目目录启动本地服务器：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

手机安装使用时，应部署到支持 HTTPS 的静态网站，例如 GitHub Pages 或 Cloudflare Pages。

## 进度存储

进度保存在浏览器 `localStorage` 中。只要使用同一浏览器和同一网站地址，关闭页面或重启手机后仍能恢复。清除浏览器网站数据、换浏览器或换设备会丢失本地进度，因此建议定期使用“导出进度”。

AI 配置也保存在当前浏览器本地，仅建议在自己的设备上填写 API Key。不填写时，补漏功能会使用题库内置的离线提示。
