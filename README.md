# 钢的学习页 | GuZhi223.github.io

这里是我的个人主页，也是一个公开的数字试验田。

我会把一些网页练习、轻量小工具、课程相关归档、开源仓库入口，以及用 AI 辅助做出来的 idea 放在这里。它不是什么包装得很完整的产品站，更像是我一边学 C/C++、数据结构、Git 和前端，一边把能跑起来、能复盘、也许对别人有点用的东西收拾到一起。

站点地址：[https://guzhi223.github.io](https://guzhi223.github.io)

## 这里有什么

### 个人主页

主入口是 `index.html`，目前包含：

- 关于我：大一计科在读，喜欢折腾，慢慢成为嘿壳。
- 一些分享：静态网页、小工具、课程拓展和从互联网上整理来的页面合集。
- 开源库：放一些我在 GitHub 上维护或学习的仓库入口。
- 传送门：GitHub 主页和邮箱入口。

### 一些分享

- `assignments/`：课程作业、外部文档和后续学习记录的归档入口。
- `projects/nutrition/`：营养素计算器，根据生活化减脂思路估算碳水、蛋白质和脂肪目标。
- `projects/particles/`：3D 手势塔罗牌抽取页面，使用 Three.js 和 MediaPipe Hands 做全屏粒子牌阵与手势交互。
- `projects/love-gallery/`：从互联网上收集整理的“低脂”爱心 HTML 展示合集，已经清理无关公众号、红包和外部推广入口。
- `projects/halloween/`、`projects/love/`：早期页面与实验留档。

### 开源项目入口

主页里也放了我目前比较想展示的几个仓库：

- [hnjingsai_help](https://github.com/GuZhi223/hnjingsai_help)：针对 `henanjingsai.cn` 的油猴脚本和题库整理。
- [chaoxing-GUI-Helper](https://github.com/GuZhi223/chaoxing-GUI-Helper)：给学习通/超星自动化流程套上的图形化启动器。
- [ono](https://github.com/GuZhi223/ono)：fork 来学习的 QQNT Xposed 模块，主要用于阅读 Android 模块结构、Hook 思路和大型开源项目组织方式。

## 技术栈

这个仓库目前尽量保持简单：

- HTML / CSS / JavaScript
- GitHub Pages 静态部署
- Three.js
- MediaPipe Hands
- 少量外部 CDN 资源

没有复杂构建流程，也没有必须安装的依赖。大部分页面直接作为静态 HTML 运行。

## 关于 AI / Vibe Coding

这个仓库里有不少内容是 AI 辅助完成的，也有很多页面是从“我想要一个东西”开始，一点点试、改、调、部署出来的。

我不想把这些项目包装成完全成熟的作品。它们有些地方还粗糙，有些代码也带着明显的学习痕迹，但它们确实记录了我怎么把一个想法从 idea 推到能打开、能交互、能继续迭代。

AI 对我来说更像代码搭档：帮我拆需求、读仓库、改页面、查问题、补适配。但最终我还是希望自己能看懂每次改动背后的逻辑，把“能跑”慢慢变成“写得更清楚”。
